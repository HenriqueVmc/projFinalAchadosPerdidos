Sistema de achados e perdidos
O que?
    Um sistema para divulgação de achados e perdidos do ifsc
Por que?
    Pois boa parte dos estudantes não moram na região
    e portanto quando perdem seus pertences, precisam ir
    até a instituição para encontrarem. Com um sistema web,
    seria possivel consultar os objetos perdidos de qualquer
    lugar e em qualquer momento.
Como?
    Por meio de uma plataforma web

Entidades
    Objetos;
        nome;
        cor;
        tipo de objetos;
        descricao
        situacao;(achado/perdido/devolvido)

    Usuários;
        nome;
        login;
        senha;
        contato;
        endereço;                        
        perfilId;
        permissaoId;

    Postagens;
        titulo;
        descricao;
        imagem;                
        usuario;
        dtPublicacao
        Objetos;
        Local;
        reconpensa
        
        
Requisitos funcionais:
    O sistema deve permitir o auto-cadastro por usuários
    O sistema deve permitir o cadastro de objetos(achados/perdidos)
    O sistema deve permitir a publicação de um post com detalhes do objeto
    O sistema deve permitir consultar os objetos publicados por usuários cadastrados
    O sistema deve disponibilizar diversos filtros para busca de objetos publicados    
    O sistema deve permitir um chat privado entre usuários
    O sistema deve permitir consultar informações do perfil do usuário
    O sistema deve permitir alterar de informações do perfil do usuário
    O sistema deve permitir consultar posts publicados pelo usuário
    O sistema deve permitir alterar posts publicados pelo usuário.

Requisitos não funcionais:
    O sistema deve zelar pela usabilidade e acessibilidade
    O sistema deve ter bom desempenho ao responder requisições
    O sistema deve zelar pela segurança das informações
    O sistema deve ser responsivo

Regras de Negócio:        
    O sistema deve notificar e solicitar a situação do objeto caso o post ultrapasse 1 mês de publicação.
    O sistema deve remover automaticamente posts que ultrapasarem 2 meses de publicação.
    O sistema deve remover os posts atualizados com situação Encontrado/Devolvido.
    O sistema deve permitir a consulta por publicações por usuários não cadastrados    
    O sistema não deve permitir o cadastro de objetos por usuários não cadastrados
    O sistema não deve permitir a publicação de posts por usuários não cadastrados
    O sistema não deve permitir o chat privado por usuários não cadastrados
